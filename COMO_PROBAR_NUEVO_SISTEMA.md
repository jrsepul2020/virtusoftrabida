# Nota: Sección de prueba eliminada

La sección que explicaba cómo probar el "Nuevo Sistema" ha sido eliminada del proyecto.

Si necesitas recuperar el panel de pruebas o los archivos asociados más adelante (hooks, helpers o componente de prueba), dímelo y los podré restaurar o mover a una rama separada.

Para pruebas normales de la aplicación sigue estos pasos rápidos:

1. Inicia el servidor de desarrollo:

```bash
npm run dev
```

2. Abre la app en el navegador:

`http://localhost:3000/`

3. Accede al panel de administración y usa las pantallas existentes (`Empresas`, `Muestras`, `Estadísticas`).

Si quieres que vuelva a crear una guía de pruebas simplificada (sin el sistema experimental), te la preparo en un momento.
